# exr_leeswijzer
Dit project is het afstudeerproject van Emilio Renders. De leeswijzer is gemaakt binnen een online omgeving. <br>
In de map exr_leeswijzer bevind zich de code voor de leeswijzer en de bewijslasten in dezelfde mappen structuur als aangegeven op de website.<br>
De bewijslasten staan naast in een goed georganiseerde mappen structuur ook er nog eens los is. Dit is zodat de de links naar de PDF in de website zelf ook kunnen werken.<br>
De leeswijzer is op de volgende URL te bekijken: https://emilioxrenders.github.io/exr_leeswijzer/.
